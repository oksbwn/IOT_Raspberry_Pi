# IOT_Raspberry_Pi
A project to demonstrate IOT on Raspberry Pi by using JAVA.

## Dependencies:
  * [Pi4J](http://bit.ly/2j32blF)
  * [Unirest](http://bit.ly/2rFQ2t3)
  * [Eclipse PAHO](http://bit.ly/2srfxPE)
